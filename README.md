# 복슬은퇴 계산기

## 1. 주요기능

- 내 자산, 저축금액, 투자 수익률을 통해 은퇴후 사용가능한 지출 계산

## 2. 실행 및 빌드

### 2.1. 초기셋팅

```shell
$ npm install
```

### 2.2. 실행 방법

```shell
$ npm start
```

### 2.3. 빌드 방법

```shell
$ npm run build
```

- 빌드 결과는 `` 폴더에 생성됨
